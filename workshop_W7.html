W7: Figurative Language Processing (FLP)

Beata Beigman Klebanov, Ekaterina Shutova, Patricia Lichtenstein, Smaranda Muresan, Anna Feldman, Chee Wee (Ben) Leong, Debanjan Ghosh

Description Schedule External Website
Live Session 1: Jul 9 (13:00-20:30 GMT)
Live Session 2 (Invited talk by Marilyn Walker): Jul 9 (21:00-22:00 GMT)
The 2nd Workshop on Figurative Language Processing aims to discuss computational approaches to figurative language (e.g., metaphors, puns, sarcasm). Shared tasks: one on metaphor and one on sarcasm detection.
